$ ./bin/sbctl user init --help
Initialize a new User with a generated auth secret

Usage:
  sbctl user init <name> [flags]

Flags:
  -h, --help               help for init
  -n, --namespace string   Kubernetes namespace (default "default")
      --protocol string    Proxy protocol (hysteria2|vless|trojan|socks5|http) (default "hysteria2")

Global Flags:
      --kubeconfig string   Path to kubeconfig file (default: ~/.kube/config)
