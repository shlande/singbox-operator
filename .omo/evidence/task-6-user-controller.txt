Added exported function `IsNodeAllowed` at end of file (lines 558-569):

```go
// IsNodeAllowed reports whether nodeName passes UserGroup allow/deny rules.
// allowedSet: if non-empty, nodeName must be present to be allowed.
// deniedSet: if nodeName is present, it is denied regardless of allowedSet.
// Deny-wins: a node in both sets is denied.
func IsNodeAllowed(nodeName string, allowedSet, deniedSet map[string]bool) bool {
    if deniedSet[nodeName] {
        return false
    }
    if len(allowedSet) > 0 && !allowedSet[nodeName] {
        return false
    }
    return true
}
```
