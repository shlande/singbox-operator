### Reconcile() Logic
1. Fetches UserGroup by NamespacedName; logs and continues on NotFound (deleted case)
2. Lists all Users in the namespace with client.MatchingFields{"spec.userGroupRef": req.Name}
3. For each User, patches annotation "singboxoperator.shlande.top/reconcile-trigger" with RFC3339 timestamp
4. Uses r.Patch() with client.MergeFrom(user.DeepCopy()) — NOT r.Update()
5. Returns ctrl.Result{}, nil

### SetupWithManager()
Registers controller with ctrl.NewControllerManagedBy(mgr).For(&proxyv1alpha1.UserGroup{}).Named("usergroup").Complete(r)

### Annotation Key
singboxoperator.shlande.top/reconcile-trigger  (matches user_controller.go pattern)

### Timestamp Format
time.Now().Format(time.RFC3339)
