### 1. Input struct — UserNodeRestrictions field added (~line 43)

```go
// UserNodeRestrictions maps userName → set of denied SingBoxNode names.
// A nil or missing entry means no restrictions (allow all nodes).
// Only the denied set is stored; the inbound pre-filter in the controller
// handles allowlist checks before the config engine is called.
UserNodeRestrictions map[string]map[string]bool
```

### 2. IsNodeAllowed exported function added (~line 188)
Signature: func IsNodeAllowed(nodeName string, allowedNodes, deniedNodes map[string]bool) bool
Semantics (deny-wins):
  1. deniedNodes[nodeName] == true  → return false
  2. len(allowedNodes) == 0         → return true (empty allowlist = allow all)
  3. allowedNodes[nodeName] == true → return true
  4. otherwise                      → return false
Nil maps are safe (map[nil][key] == zero value = false).

NOTE: A pre-existing duplicate `IsNodeAllowed` was found at the end of the file
(lines 607-619, from a prior session). It was removed to fix the redeclaration
compile error. The canonical implementation is the one added near the public API
section (~line 188).
